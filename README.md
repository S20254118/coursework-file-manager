# File Manager System

Python OOP coursework project implementing file operations with undo/redo functionality.

## Features
- Create file
- Write to file
- Read file
- Rename file
- Delete file
- List files
- Undo/Redo operations

## OOP Concepts Used
- Encapsulation
- Abstraction
- Inheritance
- Polymorphism

## Design Pattern
- Command Pattern (used for undo/redo functionality)

## Run
python main.py

## Tests
python -m unittest discover -s tests
